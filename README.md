# Setup:
All you need is to run these commands:
```bash
composer install
cp .env.example .env
php artisan key:generate
npm install
npm run dev
php artisan storage:link
sudo chmod 777 storage/app/public/ 
php artisan migrate:refresh --seed
php artisan passport:install
php artisan serve
```
